
# Java, Spring & Microservices Projects Repository

## 📌 Repository Overview

This repository is a **comprehensive collection of Java, Spring Framework, Spring Boot, Hibernate, JSP, Servlet, REST API, and Microservices projects**.  
It demonstrates hands‑on implementations of **core Java EE concepts**, **ORM**, **MVC**, **Spring ecosystem**, **RESTful services**, **Spring Cloud**, and **Reactive Programming**.

The projects are organized as **independent modules**, each focusing on a specific concept or technology widely used in **enterprise‑level applications**.

## ▶️ How to Run Projects

### General Steps

1.  Import project into **Eclipse / IntelliJ / STS**
2.  Update **Maven dependencies**
3.  Configure database (MySQL / H2 where required)
4.  Run using:
    *   `@SpringBootApplication` (Spring Boot)
    *   Server deployment (Servlet/JSP projects)
